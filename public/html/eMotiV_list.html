eMotiV 코딩리스트

본 코딩리스트는 IE9이상, 크롬, 파이어폭스, 사파리등 표준 브라우저에 최적화되어있습니다.

개발자 알림

  • id value 중 ui~, pb~로 시작하는 것은 수정하지 말아주세요.
    그외 id는 수정이 가능하며 ui~, pb~로 시작하는 것의 수정이 필요할 경우 퍼블리셔와 상의해주세요.
  • class value 및 css파일은 수정을 불가합니다.
  • markup의 위치를 임의 변경 하지 말아주세요.
    논리적으로 markup된것 이므로 수정을 해야 할 경우 퍼블리셔와 상의해주세요.

퍼블리셔 알림

  • 모바일용 html파일은 PC 크롬, 사파리브라우저에서 작업 및 확인 하며, 최종 작업물은 mobile에서 화면 확인을 기준으로 함
  • 크로스device 범위는 선정된 device를 기준으로 함
  • 수정이 발생될 경우 수정이력에 수정날짜 및 수정내용을 명시
  • 작업전 none - 작업시작전
  • 작업중 ing - 작업자가 퍼블리싱이 시작된 이후 부터 작업 완료 전까지
  • 작업완료 done - 작업자가 wireframe/디자인을 적용 후 크로스device 체크 까지 완료
  • 수정 edit - 개발자에게 배포 이후 수정 작업
  • 전체 페이지 작업중 작업완료 배포 완료 후 수정
